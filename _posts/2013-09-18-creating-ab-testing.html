  • Prerequisites:
  • You have already registered and integrated the Vessel SDK in the Application.
  1. Log into Vessel AB Dashboard and click create test.
  2. Enter the name of the test. This name needs to be unique among all tests. Something small but descriptive Eg. Landing Test
  3. You have now created a new AB test. This test is still in draft mode. You can now set the variation distribution and controls.
  4. Set Distribution and Variation Controls (Optional)
    You can change the distribution of variation for a given test. By default, it is set to 50-50% We recommend that you use this distribution to ensure that results are statistically significant. You can also add in Variation Control. Variation controls allow you associate data to each variation which will be made available to the devices.

Where to go from here:

To use your test devices to ensure that variations are working properly - Please See Using Test Devices.

Handling Exceptions

Vessel SDK allows you to report handled exceptions from your app.

For example to log handled exception you can call logException method as bellow.

{% highlight java %} try{ ....your code .... }catch(Exception ex){ // Reporting back handled exception VesselSDK.logException("Your message", ex); } {% endhighlight %}

To enable handled exception please upgrade to enterprise plan.

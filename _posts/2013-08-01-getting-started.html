Getting started with Vessel

Step 1: Download

  1. Login to Vessel Command Center
  2. Add your application
  3. Get Vessel Secret Key

Step 2: Install

  1. Download the Android SDK:
    Android SDK
  2. Unzip the downloaded vesselsdk.zip
  3. Import the vesselsdk.jar into your existing Android project, by extracting it into your "libs" folder.
  4. If your project does not already have a "libs" folder, create one at the root of the project by right-clicking the project and choosing "New" and then "Folder.

Step 3: Integrate

Your app must request the "INTERNET" and "ACCESS_NETWORK_STATE" permissions, if it isn't doing so already. Add following lines in your AndoridManifest.xml file before application tag:

{% highlight java %} {% endhighlight %}

Step 4

Add following import statement in your Activity before setContentView or in Application Context

{% highlight java %} import com.vessel.VesselSDK; public void onCreate(Bundle savedInstanceState) { super.onCreate(savedInstanceState); // Initialise Vessel VesselSDK.initialize(getApplicationContext(), secretKey); setContentView(R.layout.main); ....... //Your code goes here } {% endhighlight %}

Step 5 (Optional)

In case if you are using ProGuard for your application, then add following lines in your proguard.cfg file, It will be located inside your project folder.It will exclude VesselSDK from obfuscation process.

{% highlight java %} -dontwarn com.vessel.* -dontnote com.vessel.* -keep class com.vessel.** { *; } {% endhighlight %}

At this point you have successfully integrated VesselSDK in your application , to test the integration build and install application on your test device or on simulator. Proceed to next tutorial, enable in-app support desk.

Getting started with Vessel

  1. Setup

    1. Download framework

    2. Make sure you are using the latest version of Xcode (v4.6+) and targeting iOS 4.3 or higher.

    3. Unzip the file you downloaded and drag the Vessel.framework folder into your Xcode project folder target.

    4. Make sure the 'Copy items to destination's group folder' checkbox is checked and the target to be added to is selected.

  2. Add dependencies

    1. Select you project in Xcode

    2. Click your app name under Targets

    3. Next select the Build Phases tab

    4. Then expand Link Binary With Libraries as shown below

    5. Click the plus (+) button in the lower left of the Link Binary With Libraries.

    6. Add the following libriaes:

      • SystemConfiguration.framework
      • AdSupport.framework
      • libsqlite3.dylib

  3. Integrate

    Open your AppDelegate.m file. Import Vessel header as shown below.

    {% highlight objc %} #import <Vessel/Vessel.h> {% endhighlight %}

    To initilize Vessel Framework add the following line inside the application:didFinishLaunchingWithOptions function.

    {% highlight objc %} [Vessel initializeWithAppSecret:@"SECRET_KEY"]; {% endhighlight %}

  4. Build and run your app.

Getting started with Vessel

Step 1: Download

  1. Login to Vessel Command Center
  2. Add your application
  3. Get Vessel Secret Key
  4. Download the Android SDK and iOS framework:
    Android SDK   iOS Framework

Step 1: Add plugin to you project

{% highlight obj-c %} $ phonegap local plugin add https://github.com/vesselio/PhoneGapPlugin.git // To ensure if its added correctly execute following command $phonegap local plugin list [phonegap] com.vessel.abplugin //Soon it will be available in official plugin directory. {% endhighlight %}

Step 2: Initialize Vessel A/B Platform

After adding plugin open your main / index.html file and add

Attach Event listener on "deviceready" method as follows

{% highlight obj-c %} document.addEventListener("deviceready", onDeviceReady, false); function onDeviceReady() { // Now safe to use the PhoneGap API // Initialize Vessel A/B testing platform. You can simply initialize it once. var YOUR_SECRET_KEY = 'YOUR_APP_KEY'; vesselab.initialize( function(){console.log("VesselSDK Init done.");}, function(error){ console.log("an error occurred:" + error); }, YOUR_SECRET_KEY ); } {% endhighlight %}

Step 3: For iOS platform

In case if you experience any problem while building your application for iOS platform then verify

  1. Make sure Vessel.Framework is correctly linked, If not link it from libs folder of plugin.
  2. Make sure libsqlite3.dylib is correctly linked, If not click of Build Settings > Link Binary with Libraries and add libsqlite3.dylib.
  3. If its still giving you problem, then go to Build Settings > Compile Sources settings. And make sure VesselPlugin.m is listed their. If its missing please add VesselPlugin.m in your project.

Getting started with Vessel


Step 1: Register Application and get Secret Key

  1. Login to Vessel Command Center
  2. Add your application
  3. Get Vessel Secret Key


Step 2: Get the Android and iOS Modules

  1. Download Titanium Modules:
    Android Module   iOS Module
  2. Drag the .zip files from the download location into the project, placing it at the root of the project. Choose to "Copy Files".


Step 3: Add module entry to tiapp.xml

Open the tiapp.xml and add the following entries.

{% highlight xml %} io.vessel io.vessel {% endhighlight %}

Step 4: Initialize Vessel A/B Platform

After adding module, VesselSDK should be initialized in app.js as follows.

{% highlight javascript %} var VesselSDK = require('io.vessel'); VesselSDK.initialize(''); {% endhighlight %}

Getting started with Vessel

Step 1: Register Application and get Secret Key

  1. Login to Vessel Command Center
  2. Add your application
  3. Get Vessel Secret Key

Step 2: Install

  1. Download the Windows SDK:
    Windows SDK
  2. Add VesselSDK.dll to your project's references in Visual Studio.

Step 3: Permissions

In WMAppManifest.xml go to Capabilities tab. Make sure the following options are enabled:

{% highlight csharp %} ID_CAP_IDENTITY_DEVICE ID_CAP_NETWORKING ID_CAP_LOCATION{% endhighlight %}

Step 4: Initialize Vessel A/B Platform

Add sdk initialization to App.xaml.cs:

{% highlight csharp %} using Vessel; ... private void Application_Launching(object sender, LaunchingEventArgs e) { VesselSDK.initialize("secret_key"); } {% endhighlight %}

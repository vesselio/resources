Crash Reporting

Proguard Support

Vessel Command Center supports server side obfuscation. In case if you enabled ProGuard for your application then upload mapping.txt decode Obfuscated stacktraces. Learn more.

To upload mapping file, Login to command center select relevant application version and click on setting icon as shown bellow.

Once you upload mapping.txt file, Vessel command center decodes obfuscated statcktraces.

Crash Reporting

VesselFramework monitors and reports any unhandled exceptions. If the network is unreachable, VesselFramework saves crashes in offlne mode and reports back later to Vessel Platform.

Symbolification Support

Vessel Command Center supports server side obfuscation.. This allows the debug symbols to be stripped from the binary making the binary size smaller. For server side symbolication to work properly, you need to upload the dSYM file of the build. Refer getting started guide to enable auto uploads. Else to upload dSYM file manually refer following steps.

To upload dSYM file, login to command center select relevant application version and click on setting icon as shown below.

Once you upload dSYM file, Vessel command center starts decoding obfuscated statcktraces.

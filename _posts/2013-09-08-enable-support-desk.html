Step 2: Optional Configuration

Assign an object to the delegate property if you want to be notified when the user has finished with the interface and is ready to dismiss it. Or To set default message, use the message property as shown below.

{% highlight obj-c %} VesselInAppSupportViewController *inApp = [[VesselInAppSupportViewController alloc] init]; {inApp.message = @"Your feedback here..."; inApp.delegate = self; [self presentViewController:inApp animated:YES completion:nil];} {% endhighlight %}

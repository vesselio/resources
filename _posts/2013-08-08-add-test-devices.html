Register Test device

You can configure test devices here. This functionality is used to test different AB testing variations for internal Q/A testing. This functionality will allow you to verify all code path before you run your test.

Add Test device

  1. Install debug build of your application on emulator or device. You should have integrated VesselSDK.
  2. In log view, look for "To add this device as test device, visit https://ab.vessel.io/testdevice/activate/12343" as below.
  3. Copy and paste activation url in browser to activate this device as test device. On successful activation you can use this device to deliver specific variation. To learn more visit use test devices section.

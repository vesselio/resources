Test device

You can configure test devices here. This functionality is used to test different AB testing variations.

Add Test device

  1. If you use Eclipse for building your application then, In Eclipse
    2. {% highlight java %} Goto Window-> Show View -> Other -> Android-> Logcat. {% endhighlight %}

    Or If you use a command line for building your android application then, go to your android sdk installation directory and run following command.

    {% highlight java %} android-sdk\platform-tools>adb logcat {% endhighlight %}
  2. Install debug build of your application on emulator or device. Learn more.
  3. In log view, search for keyword "VesselSDK" and look for "To Add as test device, visit url" as bellow.
  4. Copy and paste activation url in browser to activate this device as test device. On successful activation you can use this device to deliver specific variation. To learn more visit use test devices section.

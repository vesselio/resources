Upload API's

Vessel command center provides an upload api. You can use an upload api's to directly upload and distribute builds from from your build server.

Before you use upload pi, Login to Vessel Command Center, navigate to account settings. Copy your Vessel Api Key as shown below.

To upload and distribute build

{% highlight java %} // 1. make a POST call to https://api.vesselapp.com/api3/deploy/upload/ // 2. with following paramters api_key = Your Vessel Api Key. file = File field. The apk file that needs to be distributed. releasenotes - The release notes. // 3. You can also pass: replace - Optional. 'True' if an existing version needs to be replaced. users - Optional. CSV format. Emails of users to whom the beta needs to be sent. groups - Optional. CSV format. Id's of groups to whom the beta needs to be sent. mapping - Optional. File. Proguard to decode obfuscated stacktraces. {% endhighlight %}

On successful post request, you will receive a response in json format. For example

{% highlight java %} { "push.info":{ "push.noUsersInvited":2, "push.directDownloadUrl":"http://vessel.io/NYxsasu", "push.versionName":"1.0", "push.versionCode":"5", "push.proguardEnabled":false }, "success":true } // Response parameters success - Indicates if API call was successful push.noUsersInvited - Total number of users invited. push.versionName - Version of the apk/ipa file uploaded push.versionCode - Version code/Build identifier of the apk/ipa file uploaded push.proguardEnabled - Indicates if a mapping file as uploaded push.directDownloadUrl - The url which can be used to directly download the apk/ipa file. {% endhighlight %}

In case of error, you will receive error response. For example

{% highlight java %} { "push.errors":{ "fielderrors":{ "file":[ "Beta for app Test42 with version_name 1.0 and version_code 5 already exists" ] }, "no_errors":1 }, "success":false } //In case of error response, success will be false. push.errors 足: field lists out all the errors for corresponding fields. no_errors 足 lists out the number of errors. {% endhighlight %}

You can also use a cURL utility to upload builds from command line.

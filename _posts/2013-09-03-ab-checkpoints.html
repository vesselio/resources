Reporting checkpoints

To report a checkpoint, you can use the VesselAB.checkPointVisited() method as show below

{% highlight java %} VesselAB.checkPointVisited("signedUpWithTwitter"); {% endhighlight %}

Reporting checkpoints

To report a checkpoint, you can use the VesselAB checkpointVisited method as show below:

{% highlight obj-c %} [VesselAB checkpointVisited:@"signupScreen"]; {% endhighlight %}

Reporting checkpoints

To report a checkpoint, you can use the vesselab.reportCheckpoint method as show below:

{% highlight obj-c %} // e.g. If user signUp with facebook then you can report checkpoint as follows vesselab.reportCheckpoint( function(){}, function(error){ console.log("error "+ error)}, "signUpWithFacebook"); {% endhighlight %}

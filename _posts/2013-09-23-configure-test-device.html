Configure test device

In order verify all variations or A/B test code path you can configure your test devices here. Before you configure test devices, first register device as Test device.

  1. Select the A/B test and click on configire Test devices tab.
  2. Now you can configire Test devices and ask platform deliver perticular variation.
  3. Now wait 10-15 min and reinstall your app on test device again. Test device will receive configured variation each time.
